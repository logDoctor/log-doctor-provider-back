# Infra Layer AI Instructions

## Role

Infrastructure concerns and external system connections.

## Rules

- **Structure**:
  - `infra/external`: External API clients (e.g., Azure REST client).
  - `infra/db`: Database connection factories (e.g., Cosmos DB).
- **Abstractions**: Do not implement business-specific query logic here; provided connection objects to Domain repositories.
- **Pureness**: Responsible ONLY for connectivity and raw sessions.
