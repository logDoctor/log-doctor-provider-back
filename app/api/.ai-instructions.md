# API Layer AI Instructions

## Role

Application entry point, versioning management, and request/response orchestration.

## Rules

- **Versioning**: All endpoints must be registered under a versioned router (e.g., `app/api/v1/router.py`).
- **Pattern**: Must use **Class-Based View (CBV)** pattern with `fastapi-restful`.
- **Logic**: No business logic allowed. Delegate to Domain layer.
- **DI**: Inject domain services via `dependencies.py` from the respective domain.
