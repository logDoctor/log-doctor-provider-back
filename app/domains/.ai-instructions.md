# Domain Layer AI Instructions

## Role

Core business logic and resource-driven domain models.

## Rules

- **Structure**: Each domain folder must contain:
  - `router.py`: CBV-based endpoints.
  - `schemas.py`: Pydantic models.
  - `dependencies.py`: Dependency injection providers.
  - `repository.py`: ABC interface + Concrete implementation (Cosmos, etc.).
  - `services/`: Use Case logic (e.g., `TenantStatusChecker`).
- **Naming**: Services must use `NounVerber` pattern (e.g., `SubscriptionFetcher`).
- **Isolation**: Domain services must depend on repository Interfaces (ABC), not concrete implementations.
